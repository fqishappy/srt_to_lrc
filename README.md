# srt_to_lrc

能够将双语字幕（srt格式）转换成双语歌词（lrc格式）的python脚本

#### 例如输入：

1

00:00:14,980 --> 00:00:19,040

お邪魔しまーす

打扰啦


2

00:00:19,040 --> 00:00:25,420

朝ですよ。

早上啦。

3

00:00:25,420 --> 00:00:27,920

兄さん、朝だよ。

哥哥，早上了。

#### 输出：

[00:14.98] お邪魔しまーす

[00:14.98] 打扰啦

[00:19.04] 朝ですよ。

[00:19.04] 早上啦。

[00:25.42] 兄さん、朝だよ。

[00:25.42] 哥哥，早上了。


## 使用方法

在文件所在位置打开终端

~~~ cmd
python srt_to_lrc.py input.srt output.lrc
~~~
